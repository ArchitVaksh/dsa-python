### Linear Search

**Idea:**
Check each element one by one until the target is found.

**Algorithm:**
1. Traverse the array.
2. Compare each element with the target.
3. If found, return its index.
4. If the loop finishes without finding the target, return `-1`.

**Time Complexity**
- Best Case: O(1)
- Average Case: O(n)
- Worst Case: O(n)

**Space Complexity**
- O(1)

---

### Find Maximum Element

**Idea:**
Traverse the array while keeping track of the largest element seen so far.

**Algorithm:**
1. Assume the first element is the maximum.
2. Traverse the array.
3. If the current element is greater than the current maximum, update the maximum.
4. Return the maximum after the traversal finishes.

**Time Complexity**
- Best Case: O(n)
- Average Case: O(n)
- Worst Case: O(n)

**Space Complexity**
- O(1)

---

### Find Minimum Element

**Idea:**
Traverse the array while keeping track of the smallest element seen so far.

**Algorithm:**
1. Assume the first element is the minimum.
2. Traverse the array.
3. If the current element is smaller than the current minimum, update the minimum.
4. Return the minimum after the traversal finishes.

**Time Complexity**
- Best Case: O(n)
- Average Case: O(n)
- Worst Case: O(n)

**Space Complexity**
- O(1)
