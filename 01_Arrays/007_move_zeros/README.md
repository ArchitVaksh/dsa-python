# Move All Zeros to the End

## Problem

Move all zeros to the end while maintaining the order of non-zero elements.

---

## Idea

Maintain an index where the next non-zero element should be placed.

---

## Algorithm

1. Initialize an index.
2. Traverse the array.
3. Swap non-zero elements to the current index.
4. Increment the index.

---

## Time Complexity

**O(n)**

---

## Space Complexity

**O(1)**

---

## Key Learning

- Partitioning arrays
- In-place rearrangement
